# obsidian plugin 개발 기획서
## 파일 이동 플러그인
설정 탭에서 특정 폴더를 지정하면 `현재 파일을 특정 폴더로 이동시키는 명령어`를 추가하는 플러그인
## 규칙
- 모든 UI는 obsidian 기본 API로만 구현한다.
- 모든 UX는 한국어 사용자 기준으로 구현한다.
- 절대 인라인 스타일을 사용하지 않고 obsidian native class만 사용한다.
## 구현 계획
설정 탭
- 명령어 추가 버튼을 누르면 폴더를 골라서 명령어를 추가한다.
- 명령어는 간단히 '현재 파일을 {{folder}}로 이동'이라고 이름한다.
    - 명령어의 id를 유저가 직접 만들도록 한다.
    - 툴팁에 알파벳과 `-`를 사용해서 띄어쓰기 없이 간단하게 지으라고 쓴다.
- 추가된 명령어들의 목록을 볼 수 있다.
    - 명령어 목록은 id와 삭제 버튼으로 이루어져 있다.
- 명령어를 편집하는 기능은 없고, 추가나 삭제만 가능하다.
- 설정 탭에서 추가된 명령어를 삭제하면 단축키나 명령어 팔레트에서도 검색이 되지 않도록 완전히 제거한다.
- 이동이 완료되면 나오는 알림을 껏다 킬 수 있는 옵션을 토글하는 것도 넣는다.
