# 파일 이동 플러그인

Obsidian에서 파일을 특정 폴더로 빠르게 이동시키는 명령어를 추가하는 플러그인입니다.

## 기능

- **커스텀 명령어 추가**: 설정 탭에서 원하는 폴더로 파일을 이동시키는 명령어를 생성할 수 있습니다.
- **명령어 팔레트 통합**: 추가한 명령어는 명령어 팔레트(Ctrl/Cmd + P)에서 바로 사용할 수 있습니다.
- **단축키 지원**: 각 명령어에 단축키를 할당할 수 있습니다.
- **이동 알림**: 파일 이동 완료 시 알림 표시 여부를 설정할 수 있습니다.

## 사용 방법

### 1. 명령어 추가하기

1. 설정 → 커뮤니티 플러그인 → 파일 이동 → 설정 탭으로 이동
2. "명령어 추가" 버튼 클릭
3. 명령어 ID 입력 (예: `move-to-archive`)
   - 알파벳, 숫자, 하이픈(-)만 사용 가능
   - 띄어쓰기 없이 작성
4. 이동할 대상 폴더 선택
5. 명령어가 추가됨

### 2. 명령어 사용하기

1. 이동하고 싶은 파일을 열기
2. 명령어 팔레트 열기 (Ctrl/Cmd + P)
3. "현재 파일을 {ID}로 이동" 검색
4. 선택하면 파일이 지정된 폴더로 이동됨

### 3. 명령어 삭제하기

1. 설정 탭의 "등록된 명령어 목록"에서 삭제 버튼 클릭
2. 플러그인을 다시 로드하면 명령어가 완전히 제거됨

## 설정

### 이동 완료 알림 표시
파일 이동이 완료되면 알림을 표시할지 여부를 선택할 수 있습니다.
